# Walmart Demand Forecasting with Bayesian Hyperparameter Optimisation

## Overview
This project applies machine learning to forecast daily retail demand using the Walmart M5 dataset. 
We compare grid search and Bayesian optimisation for tuning a LightGBM regression model.

## Dataset
M5 Forecasting – Accuracy (Kaggle)
Single-store subset used for computational efficiency.

## Approach
- Data preparation and feature engineering
- Baseline regression models
- Grid search hyperparameter tuning
- Bayesian hyperparameter optimisation
- Performance comparison (RMSE, MAE, MAPE)

## Repository Structure
- `data/` – raw and processed datasets
- `notebooks/` – step-by-step analysis
- `src/` – reusable scripts
- `models/` – trained models
- `reports/` – plots and outputs
